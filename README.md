## 简介

超新星思维导图是一款 **纯前端** 实现的思维导图工具，所有功能均在浏览器中运行。

- ✅ 无需服务器、无需注册、无需联网
- ✅ 所有数据在浏览器本地处理，隐私安全
- ✅ 零依赖，单个 HTML 文件即可运行
- ✅ 支持智能生成、自由编辑、多布局切换、多格式导出

---

## 核心功能

| 功能 | 说明 |
|------|------|
| ⚡ **智能生成** | 输入主题关键词或粘贴大纲文本，自动分析内容结构，生成层级分明的思维导图 |
| 🎨 **三种布局** | 放射状（头脑风暴）、树状（知识梳理）、上下（流程展示），一键切换 |
| ✏️ **自由编辑** | 双击编辑节点，Tab 添加子节点，Enter 添加兄弟节点，Delete 删除节点 |
| 🎨 **节点颜色** | 右键菜单提供 8 种预设颜色，每个节点可独立设置 |
| 📦 **丰富模板** | 内置 Python、Web 开发、AI、设计、创业、音乐、健康、阅读、会议等 9 个预设模板 |
| 📤 **多格式导出** | 支持导出为 PNG（高清 2x）、SVG（无损矢量）、Markdown（笔记兼容） |
| 🔒 **隐私安全** | 所有数据在浏览器本地处理，不上传任何服务器 |
| ↩️ **撤销支持** | 支持操作历史撤销，最多 30 步 |
